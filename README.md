# Tip Calculator

A single-screen tip and bill splitter. No build step, no dependencies.

## How to run

```bash
# Option 1 — Python (any machine with Python 3)
cd tipper
python3 -m http.server 8080
# Open http://localhost:8080

# Option 2 — just open the file directly
open index.html        # macOS
xdg-open index.html    # Linux
# Windows: double-click index.html in File Explorer
```

No installs needed. The only external resources are two Google Fonts loaded over CDN — the app works without them (fallback to system monospace/sans-serif).
